## clone コマンド（そのまま送れる）

```bash
git clone https://github.com/jeminisaga/instagram-reels-claude-workflow.git
cd instagram-reels-claude-workflow
```

Git がなくても、GitHub の **Code → Download ZIP** でもOKです。

---

## 渡さないもの（提供者側だけ）

以下は **クライアントに渡さない** でください。

| 渡さないもの | 理由 |
|-------------|------|
| `Instagram-Reels制作/` フォルダ全体 | 台帳・内部ツールが入っている |
| `クライアント台帳/` | あなた用の顧客メモ |
| `ツール/` | 案件作成スクリプト（提供者用） |
| `案件/` | 他クライアントのフォルダ |

---

## クライアントごとに分けたい場合

同じ URL を全員に渡すと、全員が **同じテンプレ** から始めます（中身は各自の PC に clone されるので混ざりません）。

**案件ごとに repo を分けたい** 場合は、提供者がクライアント名で **別リポジトリ** を作り、その URL だけ渡してください。
