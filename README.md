# Instagram Reels 制作（Claude Code）

**クライアントの方 → [★クライアントさんへ.md](★クライアントさんへ.md)**（最初に読む）

このリポジトリを **clone** し、**Claude Code** でフォルダを開いて使います。

## 初回セットアップ

```bash
git clone <このリポジトリのURL>
cd <cloneしたフォルダ>
```

1. Claude Code で **clone したフォルダ** を開く  
2. チャットに **`スタート`** と送る  
3. 質問に答える → 台本・撮影指示が `02_proposal/` に作成される  
4. 撮影 → 動画を `03_footage/raw/` へ  
5. **`編集スタート`** → `06_delivery/` に完成動画  

詳しくは [`★スタートガイド.md`](★スタートガイド.md)

## 合図

| 言葉 | タイミング |
|------|-----------|
| **スタート** | 最初の1回（ヒアリング → 提案） |
| **編集スタート** | 撮影後 |

## 必要なもの

- Claude Code が使える PC  
- 参考 Reels（**動画ファイル** または **チャットで説明**）  
- **API の追加設定は不要**

## フォルダの見方

| フォルダ | 用途 |
|---------|------|
| `00_brief/` | ヒアリング結果・参考素材 |
| `02_proposal/` | 台本・撮影指示（スタート後） |
| `03_footage/raw/` | 撮影した動画 |
| `06_delivery/` | 完成 Reels |

AI 向けの詳細指示は [`CLAUDE.md`](CLAUDE.md)（Claude Code が自動で読みます）。
