# お渡しの際にお伝えください（提供者用メモ）

このファイルは **ZIP に含めてもクライアント向け** です。必要なら削除してから渡してください。

---

## クライアントに必要なもの

- **Claude Code**（または Claude のコード実行環境）が使える PC
- **Anthropic の利用枠**（Claude Code の利用料）
- 参考にしたい **Reels の保存動画** または **内容の説明**

API キーの追加設定は **不要** です。

---

## 渡すときの一文（コピペ）

```
添付フォルダを Claude Code で開き、
チャットに「スタート」とだけ送ってください。
質問に答えると台本ができます。
撮影後は動画を 03_footage/raw/ に入れて「編集スタート」です。
```

---

## うまくいくポイント

| フェーズ | 成功のコツ |
|---------|-----------|
| スタート | 参考Reelsを **事前にスマホ保存** しておくとスムーズ |
| 撮影 | `02_proposal/script.md` を見ながら **縦9:16** で2テイク以上 |
| 編集 | PC に **ffmpeg** があると編集が進みやすい（無い場合は Claude に相談） |

---

## 想定されるつまずき

| 症状 | 対処 |
|------|------|
| スタートしても質問が来ない | フォルダごと開いているか確認。`CLAUDE.md` があるか確認 |
| 編集が進まない | `03_footage/raw/` に mp4 があるか。ffmpeg 未インストールの可能性 |
| 提案が薄い | 参考を **動画** または **秒数付きテキスト** で渡す |
